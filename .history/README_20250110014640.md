# Tailwind CSS - Coffee Style by Van Dao

## Requirements

- **nodejs**
- **yarn**
- **npm**

Clone project and run test.

```
$ git clone https://github.com/van-dao/CoffeeStyle.git
$ cd CoffeeStyle
$ yarn build
$ yarn start
```
